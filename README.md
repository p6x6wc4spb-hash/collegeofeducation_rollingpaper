#제1회 교육학부의 밤 롤링 페이퍼

## 프로젝트 소개
교육학부 행사 '교육학부의 밤'을 위해 제작한 익명 롤링페이퍼 웹앱입니다.
학우들이 익명으로 메시지를 남기면, 크리스마스 당일에 일괄 공개되는
D-day 잠금 기능을 구현.

## 주요 기능
- 익명 메시지 작성
- 크리스마스 공개일까지 D-day 카운트다운
- 공개 전/후 화면 자동 전환


## 사용 기술
- Python (Flask)
- HTML/CSS (Jinja2 템플릿)

## 개발 배경
교육학부 학생회 행사에서 실제 사용된 프로젝트로, 크리스마스 당일에 공개되느 롤링페이퍼 웹 제작
